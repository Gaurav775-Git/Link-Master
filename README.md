## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd link-master
   ```

2. **Setup Server**
   ```bash
   cd server
   npm install
   npm start
   ```

3. **Setup Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```
